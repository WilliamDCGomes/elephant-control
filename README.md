# Elephant Control

## Este aplicativo foi desenvolvido com o objetivo de gerenciar o controle de máquinas de prêmios de pelúcia. Nele existe 4 tipos de usuários diferentes, com cada um tendo acessos exclusivos para aquela conta.
## This application was developed with the purpose of managing the control of plush toy prize machines. It includes four different types of users, each having exclusive access to their respective accounts.
